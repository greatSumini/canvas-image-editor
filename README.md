# Canvas Image Editor

Sample Image Editor using HTML Canvas, Next.js

## Features

1. Adjustments

   - [ ] exposure (노출)
   - [x] brightness (밝기)
   - [ ] whites (화이트밸런스)
   - [ ] contrast (대조)
   - [ ] clarity (선명도)
   - [ ] temparature (온도)
   - [ ] gamma (색조)
   - [ ] saturation (채도)

2. Crop

   - [ ] croping ui resize
   - [ ] croping ui move
   - [ ] set croping ui rate

3. Rotate

   - [ ] rotate
   - [ ] rotating ui

## Getting Started

```bash
$ git clone https://github.com/greatSumini/canvas-image-editor
$ cd canvas-image-editor

$ yarn install

$ yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
